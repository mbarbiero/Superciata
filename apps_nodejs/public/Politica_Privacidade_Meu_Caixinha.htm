Política de Privacidade do MEU CAIXINHA

Última Atualização: 10 de janeiro de 2024

1. Introdução e quem somos

O Meu Caixinha respeita sua privacidade quanto aos dados pessoais que usa em suas funcionalidades.

Em caso de dúvida, envie um email para MeuCaixinha@smuu.com.br.

2. O que o Meu Caixinha coleta e qual a finalidade

ID do usuário de Alexa. Ela é coletada automaticamente quando você ativa a funcionalidade no app Amazon Alexa. É usada para conectar sua conta na Alexa.

Endereço de email do usuário. Permite o envio de relatórios e instruções para o email do usuário.

Fuso horário (timezone) do dispositivo. Permite ajustar os horários e datas para registro dos gastos.

3. Compartilhamento de dados pessoais com terceiros

O Meu Caixinha não compartilha dados com terceiros.

4. Armazenamento de dados e localização do tratamento

O armazenamento de dados é feito no serviço de dados da Amazon Inc, desenvolvedora da interface Alexa.

5. Por quanto tempo o Meu Caixinha mantém seus dados pessoais

Os dados de compras são mantidos por até 14 meses para que o Meu Caixinha possa emitir um relatório anual.

No entanto, o usuário pode apagar os seus dados permanentemente através de funcionalidade disponibilizada na Skill.

6. Atualização de dados pessoais?

Os dados pessoais só podem ser atualizados pelo app Alexa, disponível para Android e AppleOS.

7. Uso de Cookies/Tecnologia de Rastreamento

O Meu Caixinha não usa essas tecnologias.

8. Dados para contato
